# TCP-SERVER
Simple C++ HTTP Server

### Feature list

- [x] Multiple paths.
- [x] 404 page
- [x] No extra libraries
- [x] Multiple clients
- [ ] C++ Templating engine for HTML
- [ ] HTTPs support

# How to compile
```
gcc server.cpp -o server.exe -lws2_32 -lstdc++ -Os
```
