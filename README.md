# TCP-SERVER
Simple C++ HTTP Server

### Feature list

- [x] Multiple paths.
- [x] 404 page
- [x] No extra libraries
- [ ] C++ templating for HTML


# How to compile
```
gcc server.cpp -o server.exe -lws2_32 -lstdc++
```
